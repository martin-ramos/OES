# Quickstart

## 1. Clone OES into your project

Copy the `.opencode` directory into your repository.

## 2. Use Commands

Examples:

```
/work Add rate limit validation
/feature Implement provider fallback
/bug Fix duplicate scheduler execution
/sdd Redesign notification system
```

## 3. Always Expect

- Visible classification
- Explicit Review FINAL
- Quality Score

## 4. Activate Modes When Needed

Performance-sensitive feature:

```
Mode: performance-strict
```

Critical domain feature:

```
Mode: high-reliability
```
